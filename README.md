

**EDA = Turning numbers into a story.**

Before dashboards and models, this code is about interrogating raw data. The goal isn’t “simple reporting” — it’s uncovering hidden patterns, trends, and outliers through pure analytical logic.

**What this code does:**
1. **Explores**: Loads the dataset and checks structure, data types, missing values
2. **Analyzes**: Uses stats + visualizations to spot patterns, correlations, and anomalies  
3. **Interprets**: Transforms a static table into clear, actionable insights

Result: A clean story from raw data.
